
# 📒 Address Book in C

🚀 Project Overview

The Address Book  is a command-line application developed in C programming language to manage contact information efficiently.
It allows users to perform operations like adding, searching, updating, deleting, and viewing contacts, with data persistence using file handling.


⚙️ Features

* 👤 Add new contacts (Name, Phone Number, Email)
* 🔍 Search contacts using different criteria
* ✏️ Update existing contact details
* 🗑 Delete contacts safely
* 📋 Display all contacts in a structured format
* 💾 Save and load contacts using file storage (persistent data)


🛠️ Technologies Used

* C Programming Language
* Structures & Arrays
* Pointer
* File Handling (fopen, fread, fwrite, etc.)
* String Manipulation Functions



🔐 Input Validation
✔️ Validates phone number format
✔️ Validates email format
✔️ Handles invalid inputs gracefully
✔️ Prevents incorrect data storage


💡 Key Learnings

* Understanding structures and pointers deeply
* Working with file handling for persistent storage
* Writing modular and maintainable code
* Designing CLI-based real-world applications
* Handling edge cases and user input validation

🏆 Challenges Faced

* Managing safe update and delete operations
* Avoiding memory-related issues
* Ensing data consistency in files
* Designing a **user-friendly CLI interface
